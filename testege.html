    Gerenciar Estoque

    Use as ferramentas abaixo para gerenciar seu estoque de forma geral. Cuidado, esta ação **não pode ser desfeita**.

    Excluir Produtos do Setor:

    Selecione os produtos que deseja excluir permanentemente do cadastro (e de todos os registros de estoque/histórico). Esta ação é **irreversível**.

    Selecione o setor para o relatório de produtos

    Selecione o setor dos produtos no CSV

    Os produtos importados serão atribuídos a este setor. O arquivo deve ter as colunas: **Código**, **Descrição**, **GTIN** (opcional).

    Leitor de Código de Barras

    *É necessário HTTPS e permissão de câmera para funcionar.
